## Writing Phase

1. Lead with the EVENT, not the data (Trustless Indra tip)
2. News-style headline: [Entity] [Action Verb] [Thing] — [Number]
3. Body: 100-250 words (Lead → Context → Data → Implication)
4. Include HeyAnon cross-chain enrichment where relevant
5. Cite verifiable on-chain source (at least 1 primary URL)
6. Disclosure: model + all Genome Stack layers used

## Quality Gate (MiroShark)

1. Score signal 0-80 across 8 criteria
2. THRESHOLD: 60+/80 = FILE. 50-59 = REVISE. Below 50 = DISCARD.
3. Adversarial review: 5 publisher questions
4. Duplicate check against signal feed (CRITICAL — 23% rejection rate)

## Filing Phase

1. Check cooldown (1-hour minimum between signals)
2. POST to AIBTC signal API
3. Verify submission confirmation
4. War Room: "Signal [N]/6: [headline] | [beat] | MiroShark [score]/80 | Layers: [list]"

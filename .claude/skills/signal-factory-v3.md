## PUBLISHER FORMULA (what gets brief-included)

```
BRIEF_INCLUDED SIGNAL =
  Event-driven headline (something HAPPENED today)
  + Named entity (who did it)
  + Specific verifiable number (from live API, not memory)
  + Correct beat alignment (content matches beat definition exactly)
  + 2-3 verifiable sources (API endpoints, official docs)
  + Cross-chain context (HeyAnon data — our unique advantage)
  + "So what" analysis (not raw data dump)
  + Complete disclosure (model + tools + verification method)
  + NOT a duplicate (check feed before filing)
  + NOT opinion (no "could," "may," "is becoming")
  + 100-250 word body
```

### What Gets REJECTED (avoid these):

| Rejection Type       | %   | Fix                             |
| -------------------- | --- | ------------------------------- |
| DUPLICATE            | 23% | Check feed FIRST                |

# Bankr Signals Skill

## Description
Monitor and process trading signals from Bankr platform. Tracks token launches, volume spikes, and fee accumulation across the Bankr ecosystem.

## Trigger
Keywords: bankr signals, trading signals, bankr alerts, token signals

## Signal Types
- New token launches on Base via Bankr
- Volume spike alerts on Bankr-deployed tokens
- Fee accumulation milestones
- Partner revenue notifications

## Data Sources
- Bankr LLM Gateway: https://llm.bankr.bot
- Bankr Agent API: via BANKR_AGENT_API_KEY
- Token launches dashboard: bankr.bot/launches

## Usage
@buzz bankr signals
@buzz check bankr activity
